之前写过一篇[《你关注的股票值多少钱，不如自己算一次》](https://mp.weixin.qq.com/s?__biz=MzkwMTQ0ODE4NQ==&mid=2247484460&idx=1&sn=9ef051bb19a6da708815d3d9f0b06b24&scene=21#wechat_redirect)，当时分享了一个叫Valux的DCF估值工具。后来有不少朋友试着安装，也收到了很多反馈。

这些反馈我一直记着。最近把这个项目彻底重做了一遍，重新梳理了网页版的功能逻辑，顺手也换了个新名字 **ValueScope**，并注册了独立域名**valuescope.app**，感兴趣的朋友可以去这个链接体验下。

这篇文章算是一次更新说明，也顺便聊聊这次为什么要重做网页版。

---

## 网页版现在有 AI 了

之前的版本，AI估值只能在终端（CLI）运行，而且只有 Claude、Gemini、Qwen 这些支持CLI的模型才能用。但很多朋友并不习惯用命令行，而且把 CLI 版本部署到云端也有不少限制，网页版就一直没有接入。

这次终于补上了这块。网页版接入的是DeepSeek R1推理API。由于DeepSeek无法内置联网搜索，所以又接入了Serper（一个 Google搜索 API），实现了Google搜索信息 + 国产AI推理分析的组合。

对于网页版用户来说，这些都是安装好的，你不需要安装任何东西，打开网页就能用。

---

## 自定义估值

网页版保留了手动选择参数，自己算估值的功能。默认参数基于历史平均数据，拖动滑块可自动调整，估值结果即时更新。这个模式不需要AI推理，运行速度更快，而且完全免费，可以不限次数使用。

自定义估值和AI一键估值两种模式跑的是同一套底层模型——标准化的10年FCFF框架，包含WACC、终值和敏感性分析。区别只在于参数来源：一个是让AI推理，一个是自己定。AI估值跑完之后，参数仍然可以手动调整，估值结果会实时更新。

---

## 支持的股票市场

A股和港股的历史财务数据已经内置，免费使用，不需要额外注册API。如果你对美股或日股感兴趣，需要使用FMP（Financial Modeling Prep）的 API Key来获取历史财务数据。

FMP是一家美国的金融数据公司，我自己用了好几年。它的美股数据质量不错，而且价格相对便宜。FMP的数据主要聚焦在股票方面，尤其是财务数据很全，很适合用于快速调取财务数据和计算估值。

最近正好拿到一个合作折扣，如果通过网页版的「Financial Data API」入口中的推荐链接购买，价格会比官网便宜一些，也算是对这个项目的一点支持。具体的FMP API使用说明和折扣购买方式，也可以去GitHub项目文档里查看。

---

## 关于 AI 额度的说明

经常使用AI的朋友可能知道，API是按流量计费的，比网页端付费订阅要贵不少。如果是重度用户，使用终端版的CLI会更经济一些，但是AI引擎就无法使用像DeepSeek这样的国内大模型了，因为很少有支持CLI的版本，目前除了千问之外不多。如果只是轻度使用，网页版会更方便一些。目前网页版的AI一键估值使用的是我个人的 API key，因此只能提供限额使用。

如果额度用完还想继续用，有几个选择：

- 自定义估值模式不消耗AI额度，免费不限次，功能本身就够用；
- 自己注册DeepSeek和Serper的API，控制面板API入口填入后继续使用网页版的AI功能（单次AI分析的成本大约在1毛钱左右）；
- 如果习惯用终端，可以拉源码本地跑，接入自己的 Claude / Gemini / 千问订阅，这几个都有 CLI 版本，按月订阅通常比走API调用要划算。

## 做这个工具的初衷

ValueScope完全是个人兴趣项目，这个工具一直在我自己的投资流程里使用。之所以想把它开放出来，源于几个初心：

**🌐 技术平权** — AI时代让我们每个人都有机会靠自己把想法变成产品。专业级的数据建模和估值工具不再只属于精英机构。ValueScope想让每位散户投资者都能使用机构级的DCF的估值。

**🧱 第一性原理** — 经历这些年股市的起起落落，作为一个老股民，我始终坚信巴菲特的理念，现金流折现是价值投资的基石——基于企业未来产生的现金流，估算其真正的内在价值，这是在股市的长期生存之道。有兴趣的朋友可以看之前写的文章[《自由现金流折现估值（DCF）是价值投资者的第一性原理》](https://mp.weixin.qq.com/s?__biz=MzkwMTQ0ODE4NQ==&mid=2247484125&idx=1&sn=1a846af7252375e10078700a71fa41b6&scene=21#wechat_redirect)。

**🎯 模糊的正确 > 精确的错误** — 这个听起来跟算DCF估值有点矛盾。很多人会觉得算数没用，估值变量太多，随便调整一下变量可能差异很大。ValueScope追求的不是复杂详细的财务预测模型，而且希望抓住最核心变量，给出一个合理的估值区间，这源自Damodaran的估值方法论：关注增长、利润率、再投资这些核心驱动因素，而不是追求三张报表的精准预测和复杂勾稽。更重要的是，通过DCF估值和敏感性分析，可以反过来思考，当下市场价格隐含的核心变量是什么？你是否认同市场的预期？这个逆向思考的过程，比得到一个准确的"目标价"更有价值。
